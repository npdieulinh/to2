ĐỊNH DẠNG CSS

CSS (Cascading Style Sheets) là tập hợp các mẫu định dạng viết độc lập với mã nguồn html của trang web và dùng để định dạng cho các phần tử html tương ứng.

CSS là ngôn ngữ riêng, độc lập với HTML. Định nghĩa một lần, sử dụng nhiều lần

HTML hướng đến định hình nội dung và cấu trúc của trang web, CSS tập trung vào trình bày bố cục và hình thức trong trang web.
